This repository focuses on object-oriented programming through four projects. These are written in Java and C++ and are of increasing complexity and are focusing on different, every day use-cases.

The first project is a Java-based arithmetic calculator.

The second project is based on Java and focuses on reading a tree from a file and through different commands, we print it in JSON and DOT formats, we calculate the optimal tree through the min-max algorithm and alpha-beta pruning optimization.

The third project is the development of the classic game Connect-4 with two modes, a co-op and an AI variant. For the second one, we build the AI logic from scratch and depending of the difficulty chosen, the bot takes a different approach to the game.

The fourth project is developed in C++ and has to do with image analysis and modification. We import an image and use modifications are rotation, resize, reverse colors, balance the color histogram, grayscale and we can get information as the RGB values of each pixel.
